"Игральные кости"

Цель задания - научиться использовать оператор цикла, условный оператор, логические сравнения и научиться составлять алгоритмы.

1) Используйте специально подготовленный HTML документ
2) Создать внешний js файл и подключить его документу



2) Внутри HTML документа создать элемент div c id "result":
<div id="result"></div>


3)
Написать цикл for, который пройдет 15 итераций.



4) Итерации на которых инкремент (i) будет равен 8 или 13 необходимо пропустить.
Подобный пример был на лекции. Использовать оператор или (||)



5)
Внутри, в самом начале цикла на каждой итерации необходимо генерировать два случайных числа, 
для этого воспользуйтесь следующим кодом:
var first = Math.floor((Math.random() * 6) + 1); 
var second = Math.floor((Math.random() * 6) + 1); 
Этот код на каждой итерации цикла будет устанавливать в переменные first и second значения от 1 до 6.
Таким образом мы имитируем игральные кости.



6)
В каждой итерации цикла необходимо формировать текстовую строку из этих переменных
в формате: "Первая кость: " + first + "Вторая кость: " + second;
Затем надо будет выводить на екран значение этой строки.



7)
Выводить необходимо в елемент div, который мы заранее подготовили с помощью команды:
document.getElementById("result").innerHTML += "Первая кость: " + first + "Вторая кость: " + second + "<br>";



8)
Если переменные обе одинаковы, дополнительно вывести "Выпал дубль. Число <number>"
document.getElementById("result").innerHTML += "Выпал дубль. Число <number> <br>"
Вместо <number> подставьте значение которое было продублировано. 



9*)Если первая переменная <3 а вторая >4
Выводите сообщение "Большой разброс между костями. Разница составляет <secons - first>"
Вместо <secons - first> вычислить разницу и вывести на екран.
Использовать оператор &&
document.getElementById("result").innerHTML += "Большой разброс между костями. Разница составляет <secons - first> <br>"



10)
В каждой итерации вычислять общую сумму первой и второй переменной и прибавлять к общему количеству.

11)
Снаружи цикла (после его завершения) с помощью тернарного оператора проверить переменную total.
Если она больше 100 вывести "Победа, вы набрали <score> очков" иначе "Вы проиграли, у вас <score> очков";
Как выводить результат было показано выше.

Схема:

var total = 0
Цикл {
     вычисляем переменные каждый рас заново, 
     печатаем на экран
     печатаем сообщение про дубли если есть
     печатаем сообщение про большой разброс если есть

     total += first + second;
}
Проверяем total с помощью оператора(? :) и выводим результат.
